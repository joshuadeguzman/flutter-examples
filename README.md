## Flutter MovieHub

 A simple Flutter application that showcases movies from the TheMovieDB API

#### Overview

- Flutter Framework
- TheMovieDB API
- BLoC Pattern
- Beautiful UI
- Carousel
- Shimmer Loading Implementaion
